coding-dojo / CodingKatas
=========================

Dojo to practice [CodingKatas](http://codekata.com/).

There are other CodingKatas, such as [KataCatalogue](http://www.codingdojo.org/cgi-bin/index.pl?KataCatalogue), [Code Katas](http://www.codekatas.org/) and [Code Kata](http://katas.softwarecraftsmanship.org/). For more CodingKatas then follow this list of [Code Katas](http://www.tommycode.se/p/code-katas.html).

**Attention**: one should read [Why I Don’t Do Code Katas](https://hackhands.com/dont-code-katas/) and [The Ultimate Code Kata](http://blog.codinghorror.com/the-ultimate-code-kata/) before attempting to do code katas ( and repeat them ). Another good read is [Why should we do code katas?](https://davenicolette.wordpress.com/2012/07/25/why-should-we-do-code-katas/).
